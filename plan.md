## Phase 2: 改进计划（再执行）

### 改进原则
- **不删除原始内容** — 所有笔记保留，仅做结构化和增强
- **最小侵入** — 优先修复最影响可用性的问题
- **持续可用** — 每完成一步，仓库立即可用性提升

### Step 1: 清理（Clean）— 分离噪音

1. **归档非交易文件** — 将 `使用教程/` 下 4 个文件移至 `E:\Trading\_归档\使用教程\` 或直接删除（用户确认）
2. **归档或删除 `案件目录.md`（旅行模板）**
3. **标记空文件** — 在 7 个空/占位文件顶部添加 `> ⚠️ 待补充` 标记，或合并到统一的 `📋 待补充清单.md`

### Step 2: 修复（Fix）— 提升可读性

1. **错别字修正** — 搜索并修复已知错别字
   - `郝方向` → `好方向`（突破交易.md）
   - `有影响里的买家` → `有影响力的买家`（关键拍卖反转.md）
   - `已V型反转为例` → `以V型反转为例`
   - `博弈博弈后` → `博弈后`
2. **清理幕布格式残留** — 删除 `(空)` 占位行、`<mark style="...">` 标签、`<span style="...">` 标签
3. **图片占位替换** — 将失效的图片链接替换为带描述的占位文本：`> 📷 [图片缺失：原幕布图片 — {上下文描述}]`

### Step 3: 增强（Enhance）— 构建知识结构

1. **重写 MOC 索引文件** — 将所有空 MOC 文件（`hc_2.0.md`、`错单总结.md`、`期权.md`、`复盘.md`、`临时.md`、`使用教程.md`）替换为有实际内容的索引页，包含：
   - 文件夹内所有笔记的链接列表
   - 每篇的一句话摘要
   - 核心概念标签
2. **补全核心框架文件**：
   - **`开单先决条件.md`** — 补全空白的 L1/L2 项，添加使用说明
   - **`权重.md`** — 补全一级/二级/四级权重分类
3. **创建术语表** — 新建 `📖 术语表.md`，统一定义：
   - ZG = Zero Gamma、CW = Call Wall、PW = Put Wall
   - 压单/抬单/Sweep/毛刺结构/推保护 等特有术语
   - Delta/Gamma/GEX/DEX/OI/IV/OPEX 等期权术语
4. **创建核心概念地图** — 新建 `🗺 概念地图.md`，用 Mermaid 图或列表展示：
   - 吸收 ← → 失衡 ← → 突破
   - Delta ← → 背离 ← → 衰竭
   - GEX 正/负 ← → 震荡/趋势
   - 错单分级体系（L1/L2/L3）

### Step 4: 链接（Link）— 编织知识网络

1. **批量添加双向链接** — 为每篇笔记添加 `> 相关：[[...]] | [[...]]` 头部，建立跨文件夹关联：
   - `hc_2.0/吸收与拍卖.md` ↔ `错单总结/经典压单_抬单反转.md`（吸收概念在实盘中的应用）
   - `hc_2.0/拍卖失衡.md` ↔ `错单总结/价格跨度超级大的压单群_反转.md`
   - `期权/希腊字母.md` ↔ `临时/组合 -ES.md`（GEX 理论与实践速查）
2. **更新 `🏠 交易笔记.md` 主页** — 添加最近更新、学习路径建议

### Step 5: 归档（Archive）— 保留但不干扰

创建 `E:\Trading\_归档\` 文件夹：
- 移入使用教程（4 个文件）
- 移入案件目录（旅行模板）
- 移入纯图片复盘（4.20 US.md）

- 确认 `_归档\` 中不再包含交易相关内容
